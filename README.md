# Sistema de Atualização de Pedidos

## Executar localmente
streamlit run app.py

## Hospedagem gratuita
1. Crie conta no GitHub.
2. Envie estes arquivos para um repositório.
3. Acesse https://streamlit.io/cloud
4. Conecte o GitHub e publique o app.

Arquivos necessários:
- app.py
- requirements.txt
